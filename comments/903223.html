To the lowest level where cost and duration can be estimated and managed The purpose of creating a Work Breakdown Structure (WBS) is to decompose the project scope into smaller, more manageable components. These components are then used for estimating costs, defining schedules, assigning resources, and managing the project. The project manager should decompose the project to the lowest level where cost and duration can be estimated and managed effectively. This ensures that the project is broken down into a sufficient level of detail to accurately estimate the resources and time required for each component of the project. It also enables effective cost and schedule management throughout the project lifecycle.
