Communication pull : Les informations sont disponibles et accessibles par les destinataires lorsqu'ils en ont besoin (ex. : intranet, base de connaissances). Communication de masse : Les informations sont diffusées à un large public de manière unidirectionnelle (ex. : bulletin d'information, communiqué de presse). Communication interactive : Les informations sont échangées de manière bidirectionnelle en temps réel (ex. : réunions, discussions en ligne). Communication push : Les informations sont envoyées de manière proactive aux destinataires (ex. : e-mails, notifications).
