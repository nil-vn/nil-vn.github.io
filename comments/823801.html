A. Use a work breakdown structure (WBS) to create a project schedule based on the project and resource requirements. By using a work breakdown structure (WBS), the project manager can break down the project into smaller, manageable tasks and activities. This allows for a clear understanding of the project scope and helps in identifying potential areas of rework. By creating a project schedule based on the WBS, the project manager can allocate resources efficiently and ensure that tasks are completed in a logical sequence.
