Selected Answer: D
The team has identified the problem, i.e. task delays due to unclarity of the feature. Using the elimination technique, option A is a bad choice because the team know the impediment, there is no need to log it. Option B is e a good choice, that the Project Leaders direct the team for possible solutions. Option C is similar to option A, so it can be crossed out. Option D is the best choice, where the Project Leader empowers the team. This option is better than option B where the Project Leader directs the team.
