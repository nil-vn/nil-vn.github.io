B. Update the cost performance index (CPI) and schedule performance index (SPI) to measure the performance of the project. In preparation for the steering committee meeting, the project manager should focus on presenting quantitative performance metrics that track the project's progress, such as the Cost Performance Index (CPI) and Schedule Performance Index (SPI). These indices are widely accepted for measuring project performance in terms of cost and schedule. SPI helps assess whether the project is ahead or behind schedule, which is the focus of the meeting. CPI will also be relevant to understand cost performance, even if the meeting is focused on schedule. This prepares the project manager to present a clear, data-driven picture of the project's current state and its alignment with the overall program goals.
